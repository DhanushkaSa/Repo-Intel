# RepoMind AI — Frontend

This repository contains a Vite + React + TypeScript frontend scaffold for RepoMind AI.

Quick start:

1. Install dependencies

```bash
npm install
```

2. Run dev server

```bash
npm run dev
```

Notes:

- Tailwind is configured in `tailwind.config.cjs` and styles in `src/styles/index.css`.
- Pages are in `src/pages` and components in `src/components`.
